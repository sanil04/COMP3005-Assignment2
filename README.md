CRUD SQL Student Manager

What it does ?
This Python script makes a basic app to edit student records in a PostgreSQL database. You can look at all students, add new ones, change emails, or delete students.

What you need ?
- Python 
- psycopg2 library 
- A PostgreSQL database 

How to set up the DB
Run “Student_Setup.sql” in the pgAdmin 4 application query edition on the correct PostgreSQL database for this application.


How to run the application
- Make sure Python and the psycopg2 library are installed 
- Change the database connection info to match your server and database
- Open the code using a code editor like VS code and run it
- This can also be done on a CLI interface by running
  - Python CRUD_App.py
